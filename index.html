Deform documentation

Deform is a simple database as a service. This documentation contains all the information you need to get started using all Deform facilities.

Features

  • REST HTTP API
  • Command line interface
  • MongoDB like query language
  • Schema validation
  • Data processors
  • File storage
  • Auth tokens
  • REST Hooks
  • Web GUI (coming soon)
